Contents of the folder:

* Report_Floridia_Margelli: report of the contest work;
* selftest.s43: assembly file of the devised  selftest program;
* report_faults: fault coverage of core modules;
* result_screen: the final results obtained from TetraMAX;
* run_all_sbst.sh: bash script for running the whole testing procedure.

Running:
- replace the selftest.s43 file on the server with this one;
- place the run_all_sbst.sh script in the openmsp430_testing folder and execute it.
